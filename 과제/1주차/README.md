# LifeQuest CLI

하루 기록을 간단하게 점수와 레벨로 보여주는 Python CLI 프로그램입니다.

공부 시간, 수면 시간, 운동 여부를 입력하면 점수, 레벨, 칭호, 내일의 추천 퀘스트를 출력합니다.

## 사용한 개념

- argparse
- 변수
- 조건문
- 리스트
- 반복문
- print()



## 회고

이번 CLI 과제를 진행하면서 `argparse`를 사용해 터미널에서 값을 입력받는 방식을 처음 연습했습니다.

처음에는 `input()`으로 사용자 입력을 받는 방식이 더 익숙했지만, CLI 프로그램은 실행할 때 옵션을 함께 전달할 수 있다는 점이 새로웠습니다.

또한 조건문을 사용해 공부 시간, 수면 시간, 운동 여부에 따라 점수와 레벨이 달라지도록 만들었고, 리스트와 반복문을 사용해 추천 퀘스트를 출력해보았습니다.

아직 복잡한 기능은 없지만, Python 기초 문법을 활용해서 하나의 작은 프로그램을 완성했다는 점에서 의미가 있었습니다.
