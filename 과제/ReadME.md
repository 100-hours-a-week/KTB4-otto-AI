# LifeQuest: 현생 RPG 성장 기록 CLI

## 프로젝트 소개

LifeQuest는 하루 동안의 생활 기록을 RPG 게임처럼 분석해주는 Python CLI 프로그램입니다.

사용자는 터미널에서 이름, 직업, 공부 시간, 수면 시간, 운동 여부, 기분, 완료한 과제 수, 미룬 일 여부를 입력할 수 있습니다.  
프로그램은 입력된 값을 바탕으로 경험치, 레벨, 등급, 능력치 변화, 오늘의 칭호, 획득 배지, 내일의 추천 퀘스트를 출력합니다.

처음에는 단순히 `input()`으로 값을 입력받는 프로그램을 생각했지만, CLI 프로그램의 특징을 더 살리기 위해 `argparse`를 사용하여 터미널 명령어 옵션으로 값을 입력받도록 수정했습니다.

---

## 만든 이유

평소 하루를 돌아보는 기록은 조금 지루하게 느껴질 수 있다고 생각했습니다.  
그래서 하루의 공부, 수면, 운동, 기분, 과제 완료 여부를 RPG 게임의 성장 시스템처럼 표현하면 더 재미있게 기록할 수 있을 것 같았습니다.

또한 이번 과제를 통해 Python의 기본 문법을 실제 프로그램에 적용해보고 싶었습니다.  
단순히 문법을 따로따로 연습하는 것이 아니라, 하나의 작은 CLI 프로그램 안에서 변수, 조건문, 리스트, 반복문을 함께 사용해보는 것을 목표로 했습니다.

---

## 주요 기능

- 터미널 명령어 옵션으로 사용자 정보 입력
- 직업 선택
- 공부 시간, 수면 시간, 운동 여부, 기분, 과제 완료 수 입력
- 경험치 계산
- 레벨과 등급 계산
- 직업별 보너스 적용
- 조건에 따른 칭호 출력
- 획득 배지 출력
- 내일의 추천 퀘스트 출력

---

## 사용한 Python 개념

이번 프로젝트에서는 아래 개념을 사용했습니다.

- `argparse`
- 변수
- 자료형 변환
- 조건문 `if`, `elif`, `else`
- 비교 연산자
- 논리 연산자 `and`, `or`
- 리스트
- `append()`
- 반복문 `for`
- `print()`
- 주석

아직 함수나 클래스는 사용하지 않았습니다.  
현재는 초보 단계에서 배운 내용을 바탕으로, 코드가 위에서 아래로 순서대로 실행되도록 작성했습니다.

---

python3 lifequest.py --name 민우 --job 1 --study-hours 5 --sleep-hours 6 --exercise yes --mood 보통 --completed-tasks 2 --postponed no
