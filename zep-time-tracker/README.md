# ZEP Daily Time Tracker

ZEP 접속 시간을 자동으로 측정하기 위한 개인용 Tampermonkey 스크립트입니다.

ZEP 페이지에 접속하면 자동으로 시간이 계산되며, 사용자가 직접 `퇴근` 버튼을 눌러야 하루 기록이 마무리됩니다.  
따라서 밤 12시가 넘어도 갑자기 기록이 초기화되지 않고, 실제로 사용자가 ZEP 이용을 끝내는 시점까지 이어서 시간을 누적할 수 있습니다.

---

## 주요 기능

- ZEP 접속 시 자동으로 시간 측정 시작
- 초 단위까지 접속 시간 기록
- 컴퓨터를 껐다 켜도 기존 기록 유지
- 다른 창을 띄우거나 브라우저를 최소화해도 ZEP 탭이 열려 있으면 계속 측정
- 밤 12시가 지나도 자동 초기화되지 않음
- `퇴근` 버튼을 눌러야 해당 기록 종료
- 날짜별 기록 확인 가능
- 패널 위치 이동 가능
- GitHub에 코드 백업 가능

---


## 설치 방법

### 1. Tampermonkey 설치

Chrome 웹 스토어에서 Tampermonkey 확장 프로그램을 설치합니다.

### 2. 새 스크립트 만들기

Tampermonkey 아이콘을 클릭한 뒤 `Create a new script`를 선택합니다.

### 3. 코드 붙여넣기

기본으로 작성되어 있는 코드를 모두 삭제한 뒤, `zep-time-tracker.user.js` 파일의 코드를 전체 복사해서 붙여넣습니다.

### 4. 저장하기

`Ctrl + S`를 누르거나 메뉴에서 저장을 클릭합니다.

### 5. ZEP 접속하기

ZEP 페이지에 접속하면 오른쪽 위에 시간 측정 패널이 자동으로 표시됩니다.

---

## 사용 방법

ZEP 페이지에 접속하면 자동으로 시간이 측정됩니다.

화면에는 다음과 같은 정보가 표시됩니다.

```text
ZEP 시간 측정기
기록일 2026-05-16
1시간 25분 10초
총 5110초
상태: 계산 중
```

---

## 버튼 설명

| 버튼 | 기능 |
|---|---|
| 기록 | 날짜별 접속 기록을 확인합니다. |
| 퇴근 | 현재 기록을 종료합니다. |
| 출근 | 퇴근 후 다시 새로운 기록을 시작합니다. |
| 초기화 | 현재 기록 시간을 0으로 초기화합니다. |
| 전체삭제 | 모든 날짜의 기록을 삭제합니다. |

---

## 기록 방식

이 스크립트는 날짜가 바뀌어도 자동으로 기록을 종료하지 않습니다.

예를 들어, 5월 16일 오후에 ZEP에 접속한 뒤 밤 12시를 넘겨도 기록은 계속 `2026-05-16` 기준으로 누적됩니다.

기록은 사용자가 직접 `퇴근` 버튼을 눌렀을 때 종료됩니다.

---

## 주의사항

이 스크립트는 ZEP의 공식 출석 데이터를 가져오는 도구가 아닙니다.

브라우저에서 ZEP 페이지가 열려 있는 시간을 기준으로 개인적으로 시간을 계산합니다.

따라서 다음과 같은 경우에는 시간이 측정되지 않습니다.

- ZEP 탭을 닫은 경우
- 브라우저를 완전히 종료한 경우
- 컴퓨터가 꺼져 있는 경우
- 컴퓨터가 절전 모드에 들어간 경우

---

## 만든 이유

부트캠프 과정 중 ZEP에 머문 시간을 직접 확인하고 싶었고, 매일 학습 시간을 더 편하게 기록하기 위해 만들었습니다.

처음에는 단순히 접속 시간을 확인하는 기능으로 시작했지만, 실제 사용 중 밤 12시가 지나도 하루 기록이 이어져야 한다는 점을 고려하여 `퇴근` 버튼 방식으로 개선했습니다.
